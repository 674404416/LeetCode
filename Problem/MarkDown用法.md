



# MarkDown用法



## 1. 标题

```
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```



## 2.列表

可以用tab和shift+tab更改层级

```
- 文本1

- 文本3
```

## 3.插入图片和网址

```
Ctrl+k 插入链接
Ctrl+shift+i 插入图片
```



## 4.引用

```

```

```
- [x] 已完成项目1
  - [x] 已完成事项
  - [ ] 代办事项
- [ ] 代办项目2
- [ ] 代办项目3
```
